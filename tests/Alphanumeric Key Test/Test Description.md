# Alphanumeric Key Test

## Test Goal:
Verify that all alphanumeric keys sent by the XIAO ESP32C6 are received


## Test Materials:
- Macropad
- Macbook
- iPad
- Android

## Test Procedure:
1. Upload Alphanumeric_Key_Test.cpp to the macropad
1. Connect via BT LE to the macropad with the test device
2. Open a text editor on the test device
2. Press key 0 on the macropad
3. Ensure that the string "abcdefghijklmnopqrstuvwxyz1234567890" appears in the text editor
4. Repeat for other devices
